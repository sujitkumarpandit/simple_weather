# SkyCast Elite - Premium Weather Dashboard

A production-ready, full-stack weather application built with React, Vite, Express, and Prisma.

## Features
- **Real-time Weather**: Current conditions, hourly forecasts, and 7-day outlooks.
- **AI Summaries**: Personalized weather insights generated by Google Gemini.
- **Global Search**: Auto-complete location search powered by OpenWeather Geocoding.
- **Dynamic Backgrounds**: Immersive visual atmosphere that responds to current weather.
- **Glassmorphism UI**: Beautiful, modern interface inspired by high-end mobile weather apps.
- **Full-Stack Security**: Backend API proxying and Prisma-powered data management.

## Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion, Recharts.
- **Backend**: Node.js, Express, Prisma (ORM).
- **AI**: Google Gemini API via `@google/genai`.
- **Database**: SQLite (Dev) / PostgreSQL (Prod ready via Prisma).
- **UI Components**: Shadcn/UI (Radix primitives).

## Setup Instructions

1. **Environment Variables**:
   - Rename `.env.example` to `.env`.
   - Add your `VITE_OPENWEATHER_API_KEY` from [OpenWeather](https://openweathermap.org/api).
   - Ensure `GEMINI_API_KEY` is set in your environment or AI Studio secrets.

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Database Setup**:
   ```bash
   npx prisma generate
   npx prisma db push
   ```

4. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Architecture
- `server.ts`: Express + Vite entry point.
- `src/store/`: Zustand state management.
- `src/services/`: API integration layer.
- `src/components/`: Modular UI components.
- `prisma/`: Database schema and migrations.

## Deployment
Compatible with Vercel (frontend) and Render/Railway/Supabase (backend/DB).
Ensure all environment variables are provided in your cloud provider's dashboard.
